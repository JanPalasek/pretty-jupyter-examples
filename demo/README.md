# Demo

Shows a demonstration of capabilities of Pretty Jupyter.

## How To Run

### Short Demo
```sh
# assumption: cwd is in the root of the repository at the beginning of these commands
cd demo
jupyter nbconvert --to html --template pj short-demo.ipynb
```
